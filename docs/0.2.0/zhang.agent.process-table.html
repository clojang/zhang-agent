zhang.agent.process-table

Process-tracking for the Zhang agent.

*process-table*

dynamic

add

(add process)

Add a process to the process table.

count

(count)

Return the number of processes.

create

(create)

Create a new process table.

It is intended that there will only be one process table per agent/node.

empty?

(empty?)

Perform a boolean check on the process table; returns true if there are no processes in the table.

get-all

(get-all)

Get all processes from the process table.

lookup

(lookup id)

Lookup a process by id.

ps

(ps)

Display all the processes in an ASCII-formatted table.

remove

(remove id)

Remove the process with the given id from the process table. If a list of ids is passed instead, they will be removed from the process table.

remove-all

(remove-all)

Remove the given process ids from the process table. If not ids are given, all processes will be removed.
